# Okra Seed Vigor Dataset

This dataset contains images and labels of okra (Abelmoschus esculentus) seeds used to train a deep learning model for vigor classification.

## Contents
- `train.csv`: Training data labels
- `images/`: Folder containing training images
- `metadata.json`: Details of the dataset structure

## Usage
The dataset can be used for educational and research purposes.
